Analisis Kompleksitas

Analisis Big-O

Merge Sort Iteratif
  • Worst Case: O(n log n)
  • Best Case: O(n log n)
  • Average Case: O(n log n)
  • Space: O(n)
Merge Sort Rekursif
  • Worst Case: O(n log n)
  • Best Case: O(n log n)
  • Average Case: O(n log n)
  • Space: O(n) + stack O(log n)

Aplikasi dalam Logistik

Merge Sort digunakan dalam sistem logistik untuk:

  • Pengurutan paket berdasarkan prioritas pengiriman
  • Optimasi rute berdasarkan jarak terdekat
  • Pengelompokan paket berdasarkan berat
  • Penjadwalan pengiriman berdasarkan waktu estimasi

Rekomendasi Implementasi

Untuk Data Besar (>10.000)

Gunakan Iteratif untuk menghindari stack overflow

Untuk Performa Optimal

Gunakan Hybrid Approach (Insertion Sort untuk n kecil)

Untuk Kode yang Bersih

Gunakan Rekursif untuk readability dan maintainability
