Analisis Kompleksitas Algoritma: Merge Sort Iteratif vs Rekursif

Website analisis waktu eksekusi dan kelas kompleksitas algoritma pengurutan data.

Kategori Kompleksitas Waktu

  • O(1) — Waktu konstan
  • O(log n) — Logaritmik (divide kecil)
  • O(n) — Linear
  • O(n log n) — Linear-logaritmik (Merge Sort / Quick Sort)
  • O(n²) — Kuadratik (Bubble / Insertion)
  • O(2ⁿ) — Eksponensial
  • O(n!) — Faktorial

Untuk Merge Sort, baik versi rekursif maupun iteratif, kelas kompleksitas teoretisnya adalah: O(n log n). Namun pada implementasi dunia nyata, versi iteratif biasanya lebih cepat karena tidak ada overhead pemanggilan fungsi rekursif.
